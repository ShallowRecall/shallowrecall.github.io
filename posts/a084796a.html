📦 Git 是什么

可以把 Git 理解成一个“文件时间机器 + 协作记录器”。

几个高频概念先混个脸熟：

  • workspace：工作区，你正在改的文件
  • staging area：暂存区，准备提交的内容
  • repository：仓库，保存历史记录的地方
  • commit：一次提交
  • branch：分支
  • remote：远程仓库，比如 GitHub

💡 简化理解：工作区像草稿纸，暂存区像待寄出信封，提交就是正式存档。
